📊 Google Sheets — Base de données cloud

Une fois configuré, Google Sheets devient la source de vérité : à chaque ouverture de l'application, les données sont automatiquement chargées depuis votre Sheet. Utilisez "Envoyer vers Sheets" après chaque modification importante pour que le Sheet reste à jour.

Configuration en 3 étapes :
1. Créez un Google Sheet et copiez son ID depuis l'URL
2. Créez un Apps Script (Extensions → Apps Script) et collez le code ci-dessous
3. Déployez-le en tant que Web App (accès : Tout le monde) et copiez l'URL

✈️ Notifications Telegram

Recevez des alertes automatiques sur Telegram pour les nouvelles commandes, retards, rappels de paiement et stocks bas.

Configuration :
1. Ouvrez @BotFather sur Telegram → /newbot → copiez le Token
2. Envoyez un message à votre bot, puis ouvrez https://api.telegram.org/bot<TOKEN>/getUpdates pour obtenir votre Chat ID
